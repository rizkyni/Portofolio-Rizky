Hai, Nama saya Rizky Adhita Saputra, Saya adalah seorang web developer yang berfokus pada pengembangan front-end. Saya memiliki pengalaman dalam membangun situs web yang responsif dan menarik menggunakan HTML, CSS, dan JavaScript. Saya juga memiliki pengetahuan dasar tentang framework seperti React dan Vue.js.
Saya selalu bersemangat untuk belajar teknologi baru dan meningkatkan keterampilan saya dalam pengembangan web. Saya percaya bahwa desain yang baik dan pengalaman pengguna yang optimal adalah kunci untuk menciptakan situs web yang sukses.
